Privacy Policy

Effective Date: April 3, 2025

This Privacy Policy describes how Court Vision ("the Bot") handles information when you use it within Discord servers.

1. Information We Collect

  • Server and Channel IDs: Used to determine where to send updates and alerts.
  • User IDs: Collected temporarily when slash commands are used, for context and response routing.
  • Game Data: Pulled from the NBA’s public APIs to deliver scores, standings, and stats.

2. How We Use Your Information

We only use collected data to provide real-time NBA updates and bot functionality within your Discord server. None of this information is shared with third parties.

3. Data Storage

We do not store personal information permanently. Temporary state is cached in memory or local files to improve performance and avoid duplicate messages. No user messages or conversations are logged.

4. Data Sharing

No collected data is sold, shared, or transferred to third parties. Your server data remains private and is only used by the bot as necessary for features to function.

5. Contact

If you have any questions or would like your server’s data removed from cache, please contact us at: thecourtvisionbasketball@gmail.com

This policy may be updated periodically. Continued use of the bot implies acceptance of the latest version.
